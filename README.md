> A clock in vue fetching and displaying wikipedia's &#34;minutes to midnight&#34; from the Bulletin of the Atomic Scientists

### 🏠 [Homepage](https://pterobyte.github.io/doomsday-clock/)

## Install

```sh
yarn install
```

## Usage

```sh
yarn serve
```

## Run tests

```sh
yarn test
```
